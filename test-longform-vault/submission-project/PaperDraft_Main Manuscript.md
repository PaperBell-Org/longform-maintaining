# introduction

# Introduction

Long-form scholarly writing often spans several documents that must stay consistent with one another. The main manuscript, its supplement, the response to reviewers, and the cover letter are usually authored separately, yet they describe the same study and should carry the same publication metadata.

In this worked example we motivate the problem, state the gap our approach addresses, and outline the contributions developed in the sections that follow.
